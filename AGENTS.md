# Sobrou 2.0 App

## Stack

- React Native + Expo SDK 54
- JavaScript
- Supabase (backend/auth)

## Regras para código

- Sempre usar componentes React Native: View, Text, Pressable, ScrollView
- NUNCA usar div, className, Tailwind ou CSS web
- Animações com React Native Animated API
- Estilo sempre via StyleSheet.create()

## Skills instaladas

- design-taste-frontend: princípios de design premium adaptados para RN
- impeccable: usar /audit e /polish para revisar componentes
- emil-design-eng: animações < 300ms, easing customizado, sem ease-in
